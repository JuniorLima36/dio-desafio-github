Link para Download do Git: https://git-scm.com/download/win

Para clonar o repositorio, copie o link no GitHub e abra o Git Bash em uma pasta de sua escolha, no terminal digite os seguintes comandos.

git clone + link do repositorio
cd + caminho para pasta do repositorio adicionado 


Depois de fazer alterações ou adicionar arquivos no repositorio local, use os seguintes comandos para subir as alterações para o repositorio online.

git status 
git add .
git commit -m "Texto"
git push origin main
